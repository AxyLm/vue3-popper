# What is Vue Popper

Popper is a wrapper component for displaying popovers. It's written in Vue 3 and uses the latest version of PopperJS

# Features

  • Simple API
  • Customizable theme
  • Can display arrows
  • Up to date with Vue 3 and Popper 2
